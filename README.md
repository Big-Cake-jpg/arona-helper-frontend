## 特点

- Vue 3、Vite 5 - 极致快速
- Material You - 美观耐看
- pnpm、ESLint、Prettier - 开发便捷

## 开发

请确保电脑上安装了 Node.js 与 pnpm。

```sh
git clone https://github.com/Big-Cake-jpg/aronahelper-favorability-board.git
cd aronahelper-favorability-board
pnpm install
pnpm dev
pnpm lint
pnpm build
```
